Deskbuddy Installer

Flash Deskbuddy directly to a compatible ESP32 touchscreen board from Chrome or Edge using Web Serial.

Tested board ESP32-2432S028R CYD
Display 2.8 inch ILI9341, 240 x 320
Touch XPT2046

After flashing, use the on-device Setup page or the local web interface to configure Wi-Fi, city, brightness, and live content.
