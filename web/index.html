What It Does

Weather That Helps

Current conditions, next hours, 7-day forecast, UV, wind, and rain probability from Open-Meteo.

Workday Focus

Clock, week number, focus timer, notes, and live quote or technology headline without opening another tab.

Easy Setup

Configure Wi-Fi, city, brightness, and live content directly on the device or from the local web interface.

Install Steps

1

Connect

Plug the ESP32-2432S028R CYD into USB.

2

Install

Click Install Deskbuddy and select the serial port.

3

Wi-Fi

Open Setup on the device and start the Deskbuddy Setup portal.

4

Personalize

Use the device IP address to edit notes, widgets, city, theme, and timers.

Compatibility

Tested on ESP32-2432S028R CYD with ILI9341 240 x 320 display and XPT2046 touch. Other ESP32 touch boards may need a different TFT_eSPI setup file.
